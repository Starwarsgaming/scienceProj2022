The Process we used

CSS, HTTPS, and Python

CSS was the section of code that we used in order to make all of our text and pictures used for the website to look more presentable. It allowed us to change font size, the placement of the text, and the size and orientation of the pictures.

Github

Github is an application that we used in order to make changes to the code and still be able to share it with eachother and view the website during the editing phase.

Use of brackets

For the making of this website we decided to use the coding language called Brackets. Every thing in this website was made with the code of Brackets which was a very easy to learn and use language for coding. It is highly recommended for first time coders.
